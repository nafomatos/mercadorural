# MercadoRural 🌾

O marketplace de serviços e produtos rurais do Brasil.

## Sobre

MercadoRural conecta produtores, prestadores de serviços e compradores do agronegócio em um só lugar. Plataforma mobile-first, desenvolvida para o campo brasileiro.

## Tecnologias

- **Next.js 14** – App Router
- **TypeScript**
- **Tailwind CSS** – Mobile-first
- **Português (pt-BR)**

## Começando

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
npm run dev      # Servidor de desenvolvimento
npm run build    # Build de produção
npm run start    # Iniciar servidor de produção
npm run lint     # Verificar código
```
